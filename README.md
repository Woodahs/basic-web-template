# Basic web template
Basic web template with:
- basic HTML5 index file
- SCSS (+Compass),
- Grunt:
  - sprites
  - combine media queries (cmq) 
  - combine javascript files
  - minify CSS
  - combine JS
  - minify JS
  - JS Hint
- jQuery 1.11.2

Run "npm install" in project and "grunt watch" to watch CSS files
